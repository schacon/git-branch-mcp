# Git Branch MCP

An implementation of the Model Context Protocol (MCP) server for Git branch operations.

## Description

This project implements an MCP server that provides tools and resources for Git branch operations through the Model Context Protocol.

## Setup

```bash
# Install dependencies
npm install

# Start the server
npm start
```

## Features

- Addition tool: Simple addition function
- Greeting resource: Dynamic greeting generation

## Requirements

- Node.js
- npm
